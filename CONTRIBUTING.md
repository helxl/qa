# دليل المساهمة

## كيفية المساهمة

### 1. Fork المشروع
```bash
git clone https://github.com/helxl/qa.git
cd qa
```

### 2. إنشاء فرع للميزة
```bash
git checkout -b feature/your-feature-name
```

### 3. القيام ��التعديلات
- اتبع معايير الكود
- أضف الاختبارات
- وثق التغييرات

### 4. إرسال التعديلات
```bash
git add .
git commit -m "إضافة وصف واضح للتغيير"
git push origin feature/your-feature-name
```

### 5. إنشاء Pull Request
- اشرح التغييرات بوضوح
- أضف الصور/الفيديوهات إذا لزم الأمر
- اطلب المراجعة من فريق التطوير

## معايير الكود
- اتبع نمط الكود الموجود
- استخدم comments واضحة
- اكتب اختبارات للميزات الجديدة
- تأكد من عدم وجود أخطاء
